# FinanCity

## Coded in HTML, CSS and Vanilla JavaScript

### Your Personal AI Finance Assistant and Transactions Recorder

#### Features:
1. Interactive UI
2. Responsive Design
3. Data Importing and Exporting Options
4. AI-based Suggestions
5. Quick Diagnostics
6. Local Storage Used for Data Storage.

#### How to test it?
1. Simply go over to [FinanCity](https://financity-steel.vercel.app/);
2. Test every feature :)

#### How to run it locally?
1. Download the repository or clone it locally.
2. Add Your Hack Club AI API Key.
3. Enjoy :)
